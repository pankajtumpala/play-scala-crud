# play-scala-crud
CRUD api using Play Scala Slick

## Setup
Clone repo 

```
$ git clone https://github.com/pankajtumpala/play-scala-crud.git
```

Install sbt

```https://www.scala-sbt.org/download.html```

## Create schema
Create schema in mysql

```CREATE SCHEMA `scalacrud` ;```

## Running app

```
$ cd play-scala-crud
$ sbt run
```

## Testing

```
$ cd play-scala-crud
$ sbt test
```
